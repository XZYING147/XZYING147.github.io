Markdown编辑器——Typora的使用

2024-08-23 Software

Markdown是一种文章样式标记规则，需要有专门的文字编辑软件才可以使用。
建议使用 Typora （Typora — a markdown editor, markdown reader.）这个软件进行使用

Typora使用案例

1- 标题样式

按下Ctrl+1，可以设置1级标题样式，按下Ctrl+2，设置2级标题样式，不同标记样式按下对应Ctrl+数字就可以了。
按下Ctrl+shift+1打开【大纲视图】，还可以查看文章大纲结构。

2- 字体样式

1- Ctrl+B，文本加粗
2- Ctrl+I，斜体
3- Ctrl+U，下划线

3- 特殊公式

1- 代码。Ctrl+Shift+~
2- 超链接。Ctrl+K
3- 图片。截图后，Ctrl+V粘贴
